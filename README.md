# Project-Tech
### Server

#### installation
``` 
git clone https://github.com/Yzerrr/project-t.git
cd project-t
```
+ install node_modules
```
npm install
```
+ create a . env file
```
DB_NAME=<userName>
DB_KEY=<pass>
```
***
##### Start server

```
npm start server.js
```

```
http://localhost:3000/index
```
